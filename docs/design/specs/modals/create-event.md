### 1. Use Input Component

Current:
```tsx
<input
  id="event-date"
  type="date"
  value={field.state.value as string}
  onChange={(e) => field.handleChange(e.target.value)}
  className="w-full px-3 py-2 border border-ui-border rounded-md bg-ui-bg text-ui-text"
/>
```

Target:
```tsx
<Input
  id="event-date"
  type="date"
  value={field.state.value}
  onChange={(e) => field.handleChange(e.target.value)}
/>
```

### 2. Use Label Component

Current:
```tsx
<label className="block text-sm font-medium text-ui-text mb-1">
  <Calendar className="w-4 h-4 inline mr-1" />
  Date *
</label>
```

Target:
```tsx
<Label htmlFor="event-date" required icon={Calendar}>
  Date
</Label>
```

### 3. Use Checkbox Component

Create or use existing Checkbox component for consistent styling.

### 4. Add DialogDescription

```tsx
<DialogDescription className="sr-only">
  Form to create a new calendar event
</DialogDescription>
```
