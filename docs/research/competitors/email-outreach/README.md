## Labels

- Unlabeled: supported by public documentation or direct observation
- `[inference]`: a deduction from available evidence, but not directly stated by the source
- `[speculation]`: plausible context that is not verified by current evidence
- `[conflict]`: two sources disagree, or an older note conflicts with newer evidence

## Writing Rule

Prefer:

1. Claim
2. Evidence source
3. Confidence label when needed

Avoid:

- exact pricing if the source is not a mirrored pricing page
- exact implementation/vendor/model claims unless directly supported
- stale scale or funding numbers presented as fact

## Canonical Sources

When competitor docs and comparison docs disagree:

- competitor-specific doc wins over comparison summaries
- mirrored pricing pages win over inherited narrative notes
- comparison docs should summarize, not invent or upgrade certainty
