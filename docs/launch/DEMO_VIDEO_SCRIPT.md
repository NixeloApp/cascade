# Demo Video Script (2-3 Minutes)

Goal: a tight product walkthrough for first-time evaluators.

## Segment 1: Problem + Positioning (0:00-0:20)

- Show title card: "Nixelo: Open-source Jira + Confluence alternative".
- Narration:
  - "Nixelo combines docs and issue tracking with real-time collaboration."
  - "Self-hosted, TypeScript-first, and built for teams that want control."

## Segment 2: Fast Project Setup (0:20-0:50)

- Create a workspace and project.
- Show issue board loading.
- Narration:
  - "You can spin up a project quickly and start with Kanban or sprint workflows."

## Segment 3: Docs + Issues in One Flow (0:50-1:30)

- Open a document and type live updates.
- Link a document to an issue.
- Show issue detail with comments/subtasks.
- Narration:
  - "Docs and issues are connected, so specs and execution stay in sync."

## Segment 4: Team Operations (1:30-2:10)

- Show sprint view and dashboard metrics.
- Demonstrate assignment and status update.
- Narration:
  - "Planning, execution, and reporting live in the same workspace."

## Segment 5: Integrations + Close (2:10-2:40)

- Briefly show integrations/settings (Slack, Calendar, API keys).
- Narration:
  - "Integrate with your existing stack and keep your data in your own infra."
  - "If you want an open-source project management stack you can own, try Nixelo."

## Recording Checklist

- Use a clean seed workspace with realistic data.
- Keep cursor movement deliberate; avoid dead time.
- Use zoom/callouts only for dense UI.
- Export 1080p and include captions.
